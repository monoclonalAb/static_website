The question is solveable with a few conditional statements.

# cod<span>e</span>

```{.cpp}
class Solution {
public:
    int findClosest(int x, int y, int z) {
        return abs(x-z) == abs(y-z) ? 0 : (abs(x-z) < abs(y-z) ? 1 : 2);
    }
};
```
